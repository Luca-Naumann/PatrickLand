# 🎢 PatrickLand

> **Where Every Visit Creates Magic ✨**

PatrickLand is a modern, responsive theme park website built with HTML, CSS, and JavaScript. The project focuses on creating an immersive experience through smooth animations, interactive elements, and a premium user interface that showcases a fictional family-friendly amusement park.

# ✨ Features

### 🎡 Immersive Hero Section
- Animated Ferris wheel
- Floating clouds
- Animated sun
- Theme park skyline
- Sparkling effects
- Premium call-to-action buttons

### 📱 Fully Responsive
Designed to look great on:
- Desktop
- Laptop
- Tablet
- Mobile devices

### 🎨 Modern UI
- Clean, colourful design
- Smooth scrolling
- Glassmorphism effects
- Hover animations
- Professional typography
- Responsive layouts

### 🎠 Interactive Sections
- Hero
- About
- Attractions
- Gallery
- Reviews
- Contact
- Footer

### ⚡ Animations
- AOS (Animate On Scroll)
- CSS Keyframe animations
- Hover effects
- Floating elements
- Fade transitions
- Scroll animations

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Luca-Naumann/PatrickLand.git
```

Open the project:

```bash
cd PatrickLand
```

Simply open:

```
index.html
```

or use **VS Code Live Server** for the best development experience.
